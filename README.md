# Smart Expense Tracker

A modern web application designed to help users manage their daily expenses, monitor spending habits, and improve financial planning through a simple and user-friendly interface.

## Features

- Add and manage daily expenses
- Categorize expenses
- View spending records
- User-friendly dashboard
- Data storage with MySQL database
- Responsive design for desktop and mobile
- Real-time data handling using JSON
- Secure backend powered by Flask

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- MySQL
- JSON

## Purpose of the Project

This project was developed to provide individuals with a smart and efficient way to track expenses, reduce unnecessary spending, and improve money management.

## Installation

1. Clone the repository:

git clone https://github.com/Eliyabujilima/smart-expense-tracker.git

2. Open project folder

3. Install dependencies:

pip install -r requirements.txt

4. Configure MySQL database

5. Run the application:

python app.py
